Развёртывание

Контур развёртывания KSO Runtime на Linux КСО. Компоненты, контракты, инструменты.

Компоненты KSO Runtime

State Adapter

UKM 4 State Adapter — пишет kso_state.json, fail-closed (unknown). Поддерживает static и file source.

Sidecar Agent

Синхронизация manifest/media с backend, отправка Proof of Play, daemon-режим.

KSO Player

Chromium kiosk — показ рекламы. Runtime daemon, safety gate (idle-only play), completed PoP.

Инструменты развёртывания

Bootstrap

Безопасный установщик: dry-run → apply, копирует systemd units и env examples. Не запускает сервисы.

Preflight

Read-only валидатор готовности: каталоги, units, env, Chromium, player_shell, CLI. Не меняет систему.

Release Package Builder

Сборка kso-runtime-<version>.tar.gz с VERSION, MANIFEST.json, CHECKSUMS.sha256.

Документация

Pilot Runbook

Пошаговая инструкция первого ручного запуска на тестовом КСО. 13 шагов + rollback.

Release Package Contract

Структура релизного пакета, versioning, checksums, install layout, rollback contract.

UKM 4 Discovery

Безопасный readonly инструмент определения источника состояния УКМ 4.
