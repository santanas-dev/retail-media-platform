Порядок удаления

Данные не удаляются физически немедленно по истечении срока. Вместо этого применяется:

  • Деактивация — учётные записи, кампании (soft delete / archive)
  • Анонимизация — для персональных данных в бизнес-объектах
  • Очистка по расписанию — для истёкших токенов и временных данных

Массовая очистка данных выполняется только по утверждённому регламенту с предварительным резервным копированием.
