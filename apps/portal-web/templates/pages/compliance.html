Правила использования системы

1. Назначение системы

Retail Media Platform — служебная информационная система для управления рекламными размещениями в розничной сети. Доступ предоставляется исключительно уполномоченным сотрудникам для выполнения должностных обязанностей.

2. Журналирование действий

Все действия пользователей в системе регистрируются в журнале аудита, включая:

  • Вход и выход из системы
  • Создание, изменение, утверждение и удаление объектов
  • Изменение прав доступа и ролей
  • Публикация контента

Журнал аудита используется для целей информационной безопасности, расследования инцидентов и контроля соответствия регламентам.

3. Обработка данных

В системе обрабатываются следующие категории данных:

  • Учётные данные пользователей (логин) — для аутентификации и аудита
  • Роли и права доступа — для разграничения доступа
  • Деловая информация (кампании, креативы, расписания) — для выполнения бизнес-процессов

Система не предназначена для хранения и обработки специальных категорий персональных данных, биометрических данных, платёжной информации, фискальных данных и данных банковских карт.

4. Ограничения для пользователей

При работе в системе запрещается:

  • Вводить персональные данные (телефоны, адреса, email, паспортные данные) в текстовые поля (комментарии, названия, описания)
  • Использовать систему в целях, не связанных с должностными обязанностями
  • Предоставлять доступ к своей учётной записи третьим лицам
  • Пытаться обойти средства защиты информации
5. Блокировка и удаление учётных записей

При прекращении трудовых отношений или изменении должностных обязанностей учётная запись пользователя подлежит деактивации (блокировке). Полное удаление учётной записи не производится — сохраняются ссылки на созданные пользователем объекты и записи аудита.

Для деактивации обратитесь к администратору системы.

6. Сроки хранения данных

Сроки хранения данных определяются внутренними регламентами организации и требованиями законодательства Российской Федерации. Ознакомьтесь с политикой сроков хранения.
