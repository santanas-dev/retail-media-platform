## Что делать, если портал запущен старым процессом

Если портал был запущен до патча (процесс старше 28.06.2026 ~16:30), **перезапустите его на актуальном теге**:

```bash
# Проверить текущий HEAD
git log --oneline -1
# Должен быть: 6fac6a3 ✅ Fix RC0 runtime smoke issues

# Если нет — переключиться
git checkout v0.9.0-rc0-business-demo.1

# Остановить старый процесс портала
# Найти PID: ps aux | grep portal-web
# Остановить: kill <PID>

# Запустить заново
cd apps/portal-web && python3 main.py
```

После перезапуска проверить 18 страниц smoke-test'ом.
