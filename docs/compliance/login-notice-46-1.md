# Login Notice & Consent — 46.1

> Retail Media Platform — уведомление о работе с данными при входе
> Версия: 1.0 | Дата: 2026-06-29 | Этап: 46.1

## 1. Реализация

Уведомление добавлено на страницу входа (`/login`) и содержит четыре блока:

| Блок | Содержание |
|---|---|
| **Назначение системы** | Система предназначена исключительно для служебного использования |
| **Журналирование** | Все действия пользователей регистрируются в журнале аудита |
| **Персональные данные** | Запрет на ввод ПДн в текстовые поля |
| **Подтверждение** | Вход = ознакомление с правилами |

## 2. Дополнительные страницы

| URL | Содержание |
|---|---|
| `/compliance` | Полные правила использования системы |
| `/compliance/retention` | Политика сроков хранения данных |

## 3. Что НЕ написано

- ❌ «Полное согласие на обработку ПДн» — нет утверждённой юристами формы
- ❌ Юридически обязывающие формулировки
- ❌ Ссылки на конкретные статьи 152-ФЗ (без юристов)

## 4. Формулировки

Использованы формулировки «уведомление», «правила использования», «подтверждение ознакомления» — они не создают юридических обязательств, но информируют пользователя о:
- Служебном характере системы
- Факте журналирования
- Ограничениях на ввод ПДн
- Подтверждении ознакомления через факт входа

## 5. Доступность

Страницы `/compliance` и `/compliance/retention` — публичные, не требуют аутентификации. Добавлены в `PUBLIC_ROUTES`.
